# VS Code Keyboard Shortcuts for Windows

---

## General

| Shortcut | Action |
|:---|:---|
| `Ctrl + Shift + P` / `F1` | Show Command Palette |
| `Ctrl + P` | Quick Open (Go to File) |
| `Ctrl + Shift + N` | New Window |
| `Ctrl + Shift + W` | Close Window |
| `Ctrl + ,` | Open Settings |
| `Ctrl + K Ctrl + S` | Open Keyboard Shortcuts |
| `Ctrl + Shift + Alt + P` | Open Process Explorer |

## Basic Editing

| Shortcut | Action |
|:---|:---|
| `Ctrl + X` | Cut line (empty selection) |
| `Ctrl + C` | Copy line (empty selection) |
| `Ctrl + Shift + K` | Delete line |
| `Ctrl + Enter` | Insert line below |
| `Ctrl + Shift + Enter` | Insert line above |
| `Alt + Up Arrow` / `Down Arrow` | Move line up/down |
| `Shift + Alt + Up Arrow` / `Down Arrow` | Copy line up/down |
| `Ctrl + Shift + L` | Select all occurrences of current selection |
| `Ctrl + F2` | Select all occurrences of current word |
| `Ctrl + D` | Add selection to next find match |
| `Ctrl + U` | Undo last cursor operation |
| `Shift + Alt + I` | Insert cursor at end of each line selected |
| `Ctrl + /` | Toggle line comment |
| `Shift + Alt + A` | Toggle block comment |
| `Ctrl + Shift + A` | Toggle block comment (alternate) |
| `Tab` / `Shift + Tab` | Indent / Outdent line |
| `Ctrl + ]` / `[` | Indent / Outdent line |
| `Home` / `End` | Go to beginning / end of line |
| `Ctrl + Home` / `End` | Go to beginning / end of file |
| `Ctrl + Up Arrow` / `Down Arrow` | Scroll line up/down |
| `Alt + PgUp` / `PgDn` | Scroll page up/down |
| `Ctrl + Shift + [` / `]` | Fold / Unfold region |
| `Ctrl + K Ctrl + [` / `]` | Fold / Unfold all subregions |
| `Ctrl + K Ctrl + 0` / `J` | Fold / Unfold all regions |

## Tip

You can view and customize all shortcuts by pressing:
```
Ctrl + K Ctrl + S
```
Or go to **File > Preferences > Keyboard Shortcuts**.
